# 💬 Chat App - پروژه چت آنلاین

یک اپلیکیشن چت مدرن و زمان‌واقعی ساخته شده با **Next.js** و **Supabase** که قابلیت‌های پیشرفته‌ای مثل احراز هویت، نوتیفیکیشن، آپلود عکس و دارک مود رو پشتیبانی می‌کنه.

---

## 🚀 ویژگی‌ها

- ✅ ثبت‌نام و ورود کاربران (Authentication)
- 💬 چت زمان‌واقعی (Realtime Chat)
- 👤 مشاهده و ویرایش پروفایل کاربری
- 🖼️ آپلود عکس پروفایل
- 🔔 سیستم نوتیفیکیشن (با React Hot Toast)
- 👀 نمایش آخرین بازدید کاربران
- 🌓 دارک مود / لایت مود (با Context API)
- 📱 طراحی کاملاً ریسپانسیو (با Tailwind CSS)

---

## 🛠️ تکنولوژی‌ها

| بخش | تکنولوژی |
|-----|----------|
| فرانت‌اند | Next.js 16.2.9 + React 19.2.4 |
| استایل | Tailwind CSS 4 |
| مدیریت حالت | Context API + useReducer |
| بک‌اند و دیتابیس | Supabase (PostgreSQL + Realtime) |
| احراز هویت | Supabase Auth |
| نوتیفیکیشن | React Hot Toast |
| آیکون‌ها | Lucide React |
| تاریخ شمسی | Jalaali JS |
